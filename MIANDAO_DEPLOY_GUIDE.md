## 📋 一、技能说明

德胧商旅出行规划技能（Delonix Travel Planner）是面向德胧酒店集团店总、管理层及百达屋会员的**智能商旅出行规划 + 一键预订**AI助手。

**核心能力**：
- ✅ 实时生成精美HTML报告（德胧品牌色系）
- ✅ 德胧旗下酒店首选推荐（百达屋APP最优价）
- ✅ 12306实时火车票查询 + 高德路径规划 + 航班搜索
- ✅ 智能预算计算（经济/舒适/商务三档）
- ✅ 所有预订链接真实可一键跳转
- ✅ 行程时间线编排 + 目的地攻略 + 出行清单

## 🚀 二、一键部署步骤

### 方式一：直接导入SKILL.md（推荐，最简单）

#### 步骤1：获取SKILL.md文件

从以下任一渠道获取 `SKILL.md` 文件：

| 渠道 | 获取方式 |
|------|---------|
| **飞书云文档** | 联系AI助手发送文件 |
| **GitHub** | `git clone` 仓库后复制 `SKILL.md` |
| **本地文件** | 已放在 `skills/delonix-travel-planner/SKILL.md` |

#### 步骤2：在妙搭中创建技能

1. 打开 **飞书** → **妙搭（Bitable）**
2. 进入你的团队空间
3. 创建新应用 → 选择 **「自定义技能」**
4. 上传或粘贴 `SKILL.md` 内容
5. 保存并发布

#### 步骤3：开始使用

在妙搭对话框中输入：

```
帮我规划4月23-24日重庆出差，我是德胧店总，
4月23日下午有直营部会议，4月24日全天参会。
从天津出发，帮我看看飞机和高铁。
```

AI会自动：
1. 查询交通方案（实时12306+航班搜索）
2. 推荐德胧旗下酒店（百达屋APP最优价）
3. 生成完整HTML报告
4. 提供所有预订链接

### 方式二：通过MCP配置增强体验

如需启用12306实时查票和高德地图功能，配置MCP服务器：

#### 1. 配置 12306 MCP

```json
{
  "mcpServers": {
    "12306-mcp": {
      "command": "npx",
      "args": ["-y", "12306-mcp"]
    }
  }
}
```

#### 2. 配置高德地图MCP（可选）

```json
{
  "amap-mcp": {
    "type": "sse",
    "url": "https://mcp.amap.com/sse?key=你的高德WebKey"
  }
}
```

> 💡 高德Web Key免费申请：https://lbs.amap.com/ → 控制台 → 应用管理 → 创建应用

## 📱 三、使用说明

### 基础用法

| 说的内容 | AI会做什么 |
|---------|-----------|
| "帮我规划出差" | 启动完整出行规划流程 |
| "从X到Y怎么走" | 查询交通方案（飞机+高铁） |
| "帮我找X城市的酒店" | 推荐德胧旗下酒店 + OTA备选 |
| "预算多少" | 自动计算差旅预算 |
| "目的地攻略" | 天气+美食+景点+穿搭建议 |

### 高级用法

```
# 多天复杂行程
"规划5天出差：第1天北京→上海开会，第2天展馆布展，
第3天展会，第4天客户拜访，第5天返程"

# 多人出行
"我们3个人一起去杭州培训，帮我算预算"

# 紧急出行
"临时通知明天要去武汉，帮我快速规划"
```

### 对话即预订流程

```
1. 你说："帮我规划出差"
2. AI生成完整HTML报告
3. 你选酒店和交通："住名都，坐G1709"
4. AI给出预订链接：
   🏨 [百达屋APP预订] ← 首选，会员最优价
   📋 [万豪] [携程] [飞猪] ← 备选
   🚄 [12306购票] ← 火车票
```

## 🎨 四、HTML报告预览

生成的HTML报告包含以下模块：

| 模块 | 说明 |
|------|------|
| **Hero区** | 出行概述（城市+日期+天数） |
| **德胧酒店推荐** | 首选区域（⭐百达屋APP预订） |
| **交通方案对比** | 飞机vs高铁表格，推荐方案高亮 |
| **行程时间线** | 按时间排列的详细日程 |
| **预算明细表** | 交通+住宿+餐饮+总计 |
| **目的地攻略** | 天气/穿搭/美食/景点 |
| **备选预订渠道** | 万豪/携程/飞猪（标注备选） |
| **出行清单** | 携带物品Checklist |

**设计特点**：
- 🎨 德胧品牌色系（#1E3A8A蓝 + #F59E0B橙）
- 📱 响应式设计（手机/电脑均可完美显示）
- 🔗 所有链接真实可跳转（百达屋/万豪/携程/飞猪/12306/高德）
- 🖨️ 支持打印输出

## ⚠️ 五、注意事项

### 关于德胧酒店数据库

- 当前数据库包含重庆、天津、杭州、上海等城市数据
- 新城市数据通过AI实时搜索自动获取
- 如需添加新酒店，在对话中告知即可

### 关于百达屋Deep Link

- 当前使用 `betterwood.com` 官网链接，用户可正常访问预订
- 如需APP直接跳转（`bdwhotel://`），需德胧技术团队提供Deep Link文档
- **这不影响当前使用**，所有链接均已真实可跳转

### 关于预订链接

- 所有链接均为真实平台URL，非模拟链接
- 部分链接可能因酒店ID变更而失效，发现后请告知更新
- 预订前建议多平台比价

## 🔧 六、故障排除

| 问题 | 解决方案 |
|------|---------|
| AI不响应出行规划 | 检查SKILL.md是否正确导入妙搭 |
| 12306查票失败 | 检查MCP服务器是否正确配置 |
| 高德地图不显示 | 检查API Key是否有效（30万次/日免费） |
| HTML页面打不开 | 检查网络连接，HTML文件是否成功生成 |
| 酒店推荐不准确 | 提供更多细节（日期/预算/偏好）重新查询 |
